# Genome Firewall AI

A premium clinical decision support demo for antibiotic resistance prediction from bacterial genomes.

## What is included
- Next.js + TypeScript front end with a dark hospital-grade dashboard
- FastAPI backend scaffold with REST endpoints for prediction, simulation, assistant, and report flows
- Three.js genome visualization and Plotly-based analytics
- Docker and Docker Compose setup with PostgreSQL

## Run locally
1. Install Node dependencies: `npm install`
2. Start the web app: `npm run dev`
3. Start the backend separately from `backend/` with Python dependencies in `backend/requirements.txt`

## Demo note
The product is fully interactive and uses simulated, clearly labeled outputs for the demo flows until real model training and lab integration are connected.
